三. LodashJS数据处理

是一个一致性、模块化、高性能的 JavaScript 实用工具库。
https://www.lodashjs.com/

为什么选择 Lodash ？

Lodash 通过降低 array、number、objects、string 等等的使用难度从而让JavaScript 变得更简单。
Lodash 的模块化方法 非常适用于：

  1. 遍历 array、object 和 string
  2. 对值进行操作和检测
  3. 创建符合功能的函数
